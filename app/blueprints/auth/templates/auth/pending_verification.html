⚙ Configuración requerida

Para que el sistema envíe emails, necesitás configurar tu cuenta Gmail en el archivo .env:

  1. 1. Entrá a myaccount.google.com → Seguridad
  2. 2. Activá Verificación en 2 pasos
  3. 3. Buscá "Contraseñas de aplicación"
  4. 4. Generá una para "Correo" → copiá las 16 letras
  5. 5. Pegá esa contraseña en MAIL_PASSWORD del .env
  6. 6. Reiniciá el servidor con python run.py
